{{HTMLSidebar("Global_attributes")}}

slot はグローバル属性で、シャドウ DOM のシャドウツリー内にあるスロットをある要素に割り当てます。 slot 属性をもつ要素は、{{htmlattrxref("name", "slot")}} 属性の値がその slot 属性の値と一致する {{HTMLElement("slot")}} 要素により生成されたスロットに割り当てられます。

例えば、テンプレートとスロットの使用ガイドをみてください。
