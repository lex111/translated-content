XMLHttpRequest.channel は リクエストを送信するときにオブジェクトによって使用されるnsIChannel です。channelがまだ作られていない場合、これは null です。マルチパートリクエストのときは、これは最初のchannelであり、マルチパートリクエストの異なるパートのことではありません。アクセスするには、権限昇格が必要です。
