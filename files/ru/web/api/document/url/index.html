{{APIRef("DOM")}}

Возвращает строку URL документа HTML.

Синтаксис

string = document.URL

 

Пример

JavaScript

document.getElementById("url").textContent = document.URL;

HTML

<p id="urlText">
  URL:<br/>
  <span id="url">URL goes here</span>
</p>
